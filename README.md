# TW-Portfolio
Hi, I'm Dann. This repository contains a selection of technical writing samples focused on **API documentation**, **developer guides**, and **payment integration documentation**.

The goal of this portfolio is to demonstrate how I structure, write, and organize technical documentation for developer-facing products, especially in fintech and payment API environments.

> ### ℹ️ Note
>
>Some examples in this repository are based on fictional products created for portfolio purposes. They follow common API documentation patterns and do not contain confidential or proprietary information.
<p>&nbsp;</p>

## Focus areas

This portfolio includes examples related to:

- API documentation
- Developer documentation
- Payment integration guides
- Authentication guides
- Request and response examples
- Error handling documentation
- Markdown-based documentation
- Bilingual documentation in English and Spanish
